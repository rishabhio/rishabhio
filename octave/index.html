Octave Tutorial

Introduction

Octave is a high level language intended for numerical computations. It is very similar to MATLAB in syntax and operation. Code written in octave is mostly compatible with matlab hence it presents a free alternative to MATLAB.

Brief History

  • Version 1.0 was released in 1994
  • Version 4.0.0 was released on May 29, 2015.
  • Concept was conceived around 1988 but real development was started by John W. Eaton in 1992

Technical Details

  • Octave is written in C++ using the C++ standard library.
  • Octave uses an interpreter to execute the Octave scripting language.
  • Octave is extensible using dynamically loadable modules.
